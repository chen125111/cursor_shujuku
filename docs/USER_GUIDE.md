# User Guide

## Query Workflow

1) Open the query UI at `/`
2) Select gas components and confirm the combination
3) Review available ranges and temperature bounds
4) Enter temperature and run query

## Query History and Favorites

- Successful queries are recorded in history when authenticated
- Favorites can be saved for quick reuse
- History and favorites can be exported to PDF

## Batch Query

Use the batch panel to input multiple temperatures, run batch queries,
and compare pressure ranges. Results can be exported to PDF.

## Data Management

Use the admin UI to create, update, delete, and batch import records.
Exports are available in CSV, Excel, and PDF.

## Backup and Audit

Admins can manage backups and review audit logs from the admin UI.
