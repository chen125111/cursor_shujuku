# 文档索引

本目录包含面向不同角色（使用者/开发者/运维）的文档。建议阅读顺序如下：

- **项目概览与快速启动**：`../README.md`
- **用户手册（功能/界面/操作流程）**：`用户手册.md`
- **开发者文档（配置/接口/测试/部署）**：
  - `CONFIGURATION.md`：环境变量与配置说明（含 `.env.example`）
  - `API.md`：API 说明与示例（与 `/docs` 对应）
  - `DEPLOYMENT.md`：Docker/Compose 与生产部署要点
  - `TESTING.md`：单元测试与集成测试说明
- **运维清单**：`ops_checklist.md`
- **数据库迁移**：`db_migration.md`
- **质量与优化建议**：`QUALITY_AND_OPTIMIZATION.md`

> 说明：`系统代码文档.md` 为更完整的代码级文档（较长），可在需要深入理解实现时查阅。
