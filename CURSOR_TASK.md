# Cursor 多模型协作任务：完善气体水合物相平衡查询系统

## 项目概述
这是一个气体水合物相平衡查询系统，用于查询气体混合物在不同温度和压力条件下的相平衡数据。系统包含：
- 后端：FastAPI + SQLite
- 前端：HTML/CSS/JavaScript
- 数据库：存储气体组分数据
- 安全模块：认证、TOTP等
